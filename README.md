## Description

A video catalog application made with NestJS, proxy reverson with Nginx and docker-compose to automate containers.
## Installation

```bash
$ git clone https://github.com/IlgssonBraga/video-catalog.git
```

## Running the app

```bash
$ docker-compose up -d

```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
